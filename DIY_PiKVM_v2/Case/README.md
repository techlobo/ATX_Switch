#### 3D printed Case for DIY PiKVM v2 that supports single ATX operation

<p align="center">
<img src="./Images/3d_model_of_DIY_PiKVM_v2_case.png" alt="DIY PiKVM v2 3D printed case" width="500"/>
</p>


Printed in PLA the case supports:

- Pi4
- Power splitter from 8086.com
- C790 to provide HDMI-CSI video capture (and audio if required)
- ATX connector (RJ45 configured as per standard PiKVM ATX port)
- 40x40x10 fan
